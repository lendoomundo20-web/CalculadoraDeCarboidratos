I'm learning programming all by myself, so I wanted to create something truly fucntional, something more concrete to help me learning.

That's why I created this carbohydrates calculator, in which user describes his meal by putting the components (such as rice, beans, chicken) and the weight of each component. The calculator must show the ammount of carbo in each portion and in the meal at all.
Later it's possible to add new features, showing other nutrients and even displaying products and deals related to nutrition.
